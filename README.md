# Nekto.me-autoskip

**Скрипт для Nekto.me**

## Описание

Скрипт обновлен для сайта nekto.me — он сразу находит нового собеседника после того, как вас скипнули.
Также с этим скриптом можно изменить стандартный фон nekto.me на любой другой по вашему желанию.

## Как использовать

1. Перейдите по ссылке: [FunPay оффер](https://funpay.com/lots/offer?id=66631563)
2. Если у вас ещё нет аккаунта на FunPay — зарегистрируйтесь.
3. Оплатите удобным способом.
4. Получите скрипт и гайд по установке.
